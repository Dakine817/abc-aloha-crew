# HVAC Lead Gen Chatbot - Quick Deploy Guide

## Option 1: Netlify Drop (Easiest)

1. **Close your current Netlify tab**
2. **Go to:** https://app.netlify.com/drop
3. **Find folder:** `abc-aloha-crew/hvac-lead-gen-chatbot`
4. **Drag the ENTIRE folder** (not inside another folder)
5. **Paste into Netlify:** Select `abc-aloha-crew/hvac-lead-gen-chatbot`
6. **Deploy!**

## Option 2: Zip and Upload

1. **Right-click** `abc-aloha-crew/hvac-lead-gen-chatbot` folder
2. **Send to > Compressed (zipped) folder**
3. **Upload zipped file** to Netlify
4. **Netlify will unzip and deploy**

## Option 3: Just the public folder

For Netlify Drop, you can also drag:
- `abc-aloha-crew/hvac-lead-gen-chatbot/public`

But Option 1 is safer!

## After Deployment

Your site will be at:
- **Default:** `https://hvac-demo.netlify.app`
- **Custom:** `https://abcalohacrew.netlify.app` (after adding domain)

## Troubleshooting

If it fails again:
1. Close Netlify completely
2. Restart Netlify browser app
3. Try uploading the ZIP file instead
4. Check that no hidden characters in files

Let me know once deployed! 🐺
