🔧 Setup Instructions

After deploying to Netlify, add your Google Analytics tracking ID:

How to add tracking ID:

  1. Go to Google Analytics (analytics.google.com)
  2. Create free account
  3. Copy your Measurement ID: G-XXXXXXXXXX
  4. Paste into Netlify deployment
  5. Add to index.html in <head> section

🎯 Custom Tracking Events

// Track when someone visits:
gtag('event', 'page_view', {
  'page_path': '/services',
  'page_title': 'Our Services | ABC Aloha Crew'
});

// Track when someone requests service:
gtag('event', 'form_submit', {
  'event_category': 'Lead Generation',
  'event_label': 'HVAC Service Request'
});

// Track chatbot conversations:
gtag('event', 'chat_message', {
  'event_category': 'Chatbot',
  'event_label': 'User sent message'
});
