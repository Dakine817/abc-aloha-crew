# HVAC Lead Generation Chatbot Demo

## Overview
AI-powered chatbot for HVAC businesses to capture leads via website, Telegram, and social media.

## Tech Stack
- **AI Model:** Anthropic Claude (sonnet-4-6)
- **Platform:** Web + Telegram integration
- **Features:** Lead capture, FAQ, appointment scheduling, pricing inquiry

## Quick Start
1. Clone this project
2. Configure API keys in `.env`
3. Run `npm start`

## Demo Features
- ✅ 24/7 lead capture
- ✅ HVAC-specific knowledge base
- ✅ Multi-channel support (Web, Telegram, WhatsApp)
- ✅ CRM integration hooks
- ✅ Lead scoring and prioritization
